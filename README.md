# New Project

Starter repository for a WordPress plugin.

## Structure

- `new-project.php`: main plugin bootstrap file
- `includes/class-new-project.php`: plugin runtime class
- `readme.txt`: WordPress.org-style plugin metadata
- `uninstall.php`: uninstall hook

## Local development

Copy or symlink this folder into your WordPress installation's `wp-content/plugins/` directory, then activate **New Project** from the WordPress admin.

## Next steps

1. Replace the placeholder metadata with your real plugin name, URI, author, and description.
2. Add your feature code under `includes/` or additional folders as the plugin grows.
3. Create a remote repository on GitHub, GitLab, or another git host and push this repo so you can work from multiple machines.
